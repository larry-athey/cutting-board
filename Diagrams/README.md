# Cutting Board Electronics

You may notice that the printed circuit board used here isn't as fancy as the boards for my other projects. The reason being, I created it with the Fritzing program which is geared for hobbyists. If somebody else wants to create a fancier board, I'll happily accept it and replace the current one.

Keep in mind that the LilyGo T-Display S3 mounts to the bottom side of the PCB and only uses the pins on the bottom edge of the board. This is intentional to make the motor and sensor connections serviceable without the need to disasseble the whole controller unit.

The footprint for VR1 looks like a TO-220 mounting for an LM7803, but is intended for a small 3.3 volt buck regulator which are available from various sources that use the same footprint. I don't recommend using an LM7803 due to the heat that they all generate.
